## 📌 Overview

**People's Priorities** is an AI-powered constituency planning platform that helps Members of Parliament and local authorities identify and prioritize development needs using real citizen feedback.

Instead of relying only on manual surveys and scattered complaints, citizens can report issues through **voice, photos, or text** in their local language. The platform combines these reports with census and public infrastructure data to generate transparent, evidence-based development priorities displayed on an interactive map.

---

# 🎯 Problem Statement

Development planning often depends on fragmented complaints, manual surveys, and disconnected datasets.

As a result:

- Important issues remain unnoticed
- Funds are not always allocated efficiently
- Decision making becomes slow
- Citizen participation remains limited

People's Priorities addresses these challenges through AI-driven constituency intelligence.

---

# 🚀 Solution

The platform enables citizens to submit issues using voice, photos, or text.

AI understands every report, combines it with public datasets, identifies development hotspots, and generates transparent priority rankings to help governments make evidence-based decisions.
